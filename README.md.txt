# Personalized AI Emotion Adaptive Music System

## Overview
This project is an AI-powered real-time emotion detection and adaptive music recommendation system developed using Python, OpenCV, TensorFlow, and CustomTkinter.

The system detects user emotions through webcam facial analysis and automatically recommends suitable music based on emotional state.

---

## Features

- Real-time face detection
- Emotion recognition using CNN
- Personalized user login
- AI-based music recommendation
- Emotion analytics dashboard
- Wellness suggestions
- Automatic emotion tracking
- Real-time webcam processing
- Modern GUI using CustomTkinter

## Installation

### Clone Repository

```bash
git clone https://github.com/sudeep2006/AI-Emotion-Music-System.git
```

### Move to Project Folder

```bash
cd AI-Emotion-Music-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Project

```bash
python gui.py
```

---

## How It Works

1. Webcam captures real-time face data
2. OpenCV detects face region
3. CNN model predicts emotion
4. System recommends suitable music
5. Emotion logs stored for analytics
6. User receives wellness suggestions
